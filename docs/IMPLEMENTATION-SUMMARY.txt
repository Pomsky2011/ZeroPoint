================================================================================
I/O REGISTER MAP
================================================================================

Bank $D8 - I/O Registers:
  $D80000-$D8000F (16 bytes) - PPU Control
  $D80010-$D8001F (16 bytes) - APU Control
  $D80020-$D8002F (16 bytes) - DMA Control (placeholder)
  $D80040-$D80047 (8 bytes)  - Display Status (read-only)

Bank $B0 - PPU Memory Window (64 KB)
  Access PPU's 64 KiB memory directly

Bank $A0 - APU Memory Window (64 KB)
  Access APU's 64 KiB memory directly

================================================================================
ASSEMBLY USAGE EXAMPLES
================================================================================

Start PPU:
    LDA #$01
    STA $D80000         ; PPUCTRL: start

Upload to PPU memory:
    LDA #$B0
    SDB                 ; Switch to PPU bank
    LDA #$42
    STA $0000           ; Write to PPU memory

Upload to APU memory:
    LDA #$A0
    SDB                 ; Switch to APU bank
    LDA #$00
    STA $8000           ; Write to APU BIOS

Wait for V-Blank:
    wait_vblank:
        LDA $D80044     ; DISP_STATUS
        AND #$01        ; V-Blank bit
        BEQ wait_vblank
